# Arduino IDE

## Boardのインストール
IDEを開きFile > Preferences 

Additional boards manager URLs"に以下を入力し"OK"。
https://raw.githubusercontent.com/sparkfun/Arduino_Boards/master/IDE_Board_Manager/package_sparkfun_index.json


Tools > Board > Boards Manager 

"sparkfun"で検索、"SparkFun AVR Boards by SparkFun Electronics"の"INSTALL"を押す。

## IDE Version
- 2.3.3

## Board Version

- SparkFun AVR Boards 1.1.13

## Library Version

- Adafruit Neo Pixel 1.15.4

# Flash Setting
- Tools > Board > SparkFun AVR Boards > SparkFun Pro Micro
- Tools > Processor > ATmega32U4 (5V, 16MHz)
